# STATE

Active Arbiters & Auditors must be recorded in each audit block so that the entire network is aware of active arbiters and auditors.
This could be viable for several reasons beyond state management.
For state it records active miners and their activity which is vital for other miners to lookup domain and contact information for efficient communications.
Viable for rewards, statistics, and historical performance.
It allows for private verification routes shielding transactions until they are ready.
Condense receipts for prior periods into new unified receipts an optimization that will help to keep light clients more efficient and help validators more quickly confirm if funds do exist.
