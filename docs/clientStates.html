Universal Web Socket States

STATE

Value State Description
0 DEAD - NOT ACTIVATED Client object initialized and ready to begin connecting.
1 DISCOVERING Client is attempting to send discovery to get certificate.
2 DISCOVERED Client has certificate and is ready to connect.
3 CONNECTING Client is attempting to connect.
4 CONNECTED Client is connected and ready to communicate.
5 CLOSING The connection is in the process of closing.
6 CLOSED The connection is closed or couldn't be opened.
7 DESTROYING The Client object itself is being torn down.
8 DESTROYED The Client object is destroyed.

READYSTATE

The readyState attribute represents the state of the client, socket, and/or connection. The readyState property is less detailed than the state property but is made available to aid WWW developers already familiar with WebSocket ready states.

Value State Description
0 CONNECTING Socket has been created. The connection is not yet open.
1 OPEN The connection is open and ready to communicate.
2 CLOSING The connection is in the process of closing.
3 CLOSED The connection is closed or couldn't be opened.
