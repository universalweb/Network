# AGENT USE RULES

Included in the repo are basic AGENT instructions but do not use agents for or on priority code, cryptography, and or back-end components. Agents can be used for Web App replated tasks that are none priority items and or straight forward redundant tasks. A human must review any code going into the codebase.

You may use AGENTS to generally guide development to work within standards and observe wise security practice.
No VIBE coding!
